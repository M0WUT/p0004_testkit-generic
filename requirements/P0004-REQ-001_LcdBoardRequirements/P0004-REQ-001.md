LCD Board Requirements
# General
| Requirement ID | Requirement | Justification |
| --- | --- | --- |
| GEN-01 | PCB shall be designed for assembly at JLCPCB using the cheapest possible process. i.e. prefarably single-sided assembly, use parts that exist in JLC standard part library etc | To minimise cost when getting production quantity assembled |


# Electrical
| Requirement ID | Requirement | Justification |
| --- | --- | --- |
| ELEC-01 | PCB must feature an graphic LCD | To allow information to be displayed in a reconfigurable way |
| ELEC-02 | PCB must have a minimum of 4 buttons | To allow flexible user input |
| ELEC-03 | PCB should have USB host port (that is accessible without disassembly) | To allow for easy data ingress / egress |
